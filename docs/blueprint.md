# **App Name**: BPSDM Knowledge Hub

## Core Features:

- Homepage Highlights: Showcase a hero section with the KMS BPSDM's mission, along with the newest and most popular knowledge items, enhanced with dummy statistics.
- Knowledge Base: Present a list of knowledge articles with a card layout that includes the title, summary, author, and date; allow users to filter by categories and tags, and include a (non-functional) search bar.
- Detailed Article View: Display complete articles along with metadata and a dummy comment section that supports nested replies, as well as non-functional like/reply buttons.
- Discussion Forum: Simulate a modern forum with threads and nested replies.
- Learning Resources: Organize modules, documents, and links into a modern-looking grid or list with badges indicating the type of resource.
- Top Navigation Bar: Implement a sticky navigation bar at the top with links to Home, Knowledge, Discussion, Learning Resources, and About KMS.
- UI Styling and Theming: Generative AI tool suggests improvements to page styling (colors, fonts, element sizing etc), based on the visual design guidelines.

## Style Guidelines:

- Primary color: Green (#009343) for the header and main navigation, representing the identity of Jawa Barat.
- Secondary color: Blue (#00BEF0) for links and informational elements, providing a clean and informative feel.
- Accent color: Orange (#FABF25) for action buttons, highlights, and badges, drawing attention without being overwhelming.
- Headline font: 'Poppins', sans-serif, for headlines and titles, ensuring clarity and modernity.
- Body font: 'PT Sans', sans-serif, for body text, providing readability and a professional touch.
- Maintain a clean and spacious layout that's easy to navigate.
- Use simple and professional icons to enhance the user experience.
- Utilize a card layout for knowledge articles and a modern grid/list view for learning resources.
- Design a comment and reply thread similar to a modern forum.
- Include an official agency footer on all pages.
